# Diagram Exports

Source Mermaid diagrams live in this folder. To export PNGs/SVGs for presentations:

## Install mermaid-cli

```bash
npm i -D @mermaid-js/mermaid-cli
```

## Export PNGs

```bash
npm run export:diagrams
```

## Export SVGs (optional)

```bash
npm run export:diagrams:svg
```
